Hotkeys

Cmd is the key immediately left of the spacebar. On the MacBook that is Cmd. On the XPS that key is physically labelled Alt — a swap at the input layer makes it Command, so both machines are driven from the same thumb.

Caps Lock is Ctrl.

There is a rule underneath all of this, and once you have it you can stop looking things up: Cmd+letter opens a thing you look at, Cmd+Shift+letter opens one of the bar's menus. The shifted half of the keyboard changes a setting; the unshifted half shows you an app.

Apps

Keys Opens
CmdReturn Terminal
CmdN Browser
CmdShiftN Browser, private window
CmdE Files
CmdI Messages
CmdM Music
CmdShiftM Maps
CmdShiftW Weather
CmdShiftP Photos
CmdShiftD Calendar
CmdShiftC Calendar, again

These focus the app if it's already running instead of opening a second copy. CmdE is the deliberate exception: it always gives you a new window, because browsing files is something you do several of at once.

The calendar answers to two chords on purpose — D for date, C for calendar. Whichever word you reach for, the key is there.

Windows and desktops

Keys Action
CmdQ Close the window
CmdTab Next window
Cmd1–5 Go to that desktop
CmdCtrl1–5 Send the window to that desktop
CtrlLeft / CtrlRight Previous / next desktop
Three-finger swipe Previous / next desktop
CmdDelete Lock the screen
CmdEscape Log out

Five desktops, always. A sixth can appear if an app insists on one, but nothing new lands there — a window that would be stranded gets pulled back to the desktop you're actually on.

Mac shortcuts, everywhere

These are rewritten before the app ever sees them, so every app gets them: you press the Mac chord, the app receives its Linux one.

Keys Action
CmdC / CmdV / CmdX Copy / paste / cut
CmdA Select all
CmdZ Undo
CmdS Save
CmdF Find
CmdT / CmdW New tab / close tab
CmdShiftT Reopen the closed tab
CmdLeft / CmdRight Back / forward
CmdOptionI Developer tools

The terminal is deliberately left out of the rewrite, so CtrlC there is still an interrupt. It implements CmdC, CmdV, CmdT and CmdW itself.
