# Hotkeys

**Cmd is the key immediately left of the spacebar.** On the MacBook that is Cmd.
On the XPS that key is physically labelled **Alt** — a swap at the input layer
makes it Command, so both machines are driven from the same thumb.

**Caps Lock is Ctrl.**

There is a rule underneath all of this, and once you have it you can stop looking
things up: `Cmd+letter` opens a thing you look at, `Cmd+Shift+letter` opens one of
the bar's menus. The shifted half of the keyboard changes a setting; the unshifted
half shows you an app.

## Apps

| Keys | Opens |
|---|---|
| `Cmd+Return` | Terminal |
| `Cmd+N` | Browser |
| `Cmd+Shift+N` | Browser, private window |
| `Cmd+E` | Files |
| `Cmd+I` | Messages |
| `Cmd+M` | Music |
| `Cmd+Shift+M` | Maps |
| `Cmd+Shift+W` | Weather |
| `Cmd+Shift+P` | Photos |
| `Cmd+Shift+D` | Calendar |
| `Cmd+Shift+C` | Calendar, again |

These focus the app if it's already running instead of opening a second copy.
`Cmd+E` is the deliberate exception: it always gives you a new window, because
browsing files is something you do several of at once.

The calendar answers to two chords on purpose — `D` for date, `C` for calendar.
Whichever word you reach for, the key is there.

## Windows and desktops

| Keys | Action |
|---|---|
| `Cmd+Q` | Close the window |
| `Cmd+Tab` | Next window |
| `Cmd+1`–`5` | Go to that desktop |
| `Cmd+Ctrl+1`–`5` | Send the window to that desktop |
| `Ctrl+Left` / `Ctrl+Right` | Previous / next desktop |
| Three-finger swipe | Previous / next desktop |
| `Cmd+Delete` | Lock the screen |
| `Cmd+Escape` | Log out |

Five desktops, always. A sixth can appear if an app insists on one, but nothing
new lands there — a window that would be stranded gets pulled back to the desktop
you're actually on.

## Mac shortcuts, everywhere

These are rewritten before the app ever sees them, so every app gets them: you
press the Mac chord, the app receives its Linux one.

| Keys | Action |
|---|---|
| `Cmd+C` / `Cmd+V` / `Cmd+X` | Copy / paste / cut |
| `Cmd+A` | Select all |
| `Cmd+Z` | Undo |
| `Cmd+S` | Save |
| `Cmd+F` | Find |
| `Cmd+T` / `Cmd+W` | New tab / close tab |
| `Cmd+Shift+T` | Reopen the closed tab |
| `Cmd+Left` / `Cmd+Right` | Back / forward |
| `Cmd+Option+I` | Developer tools |

The terminal is deliberately left out of the rewrite, so `Ctrl+C` there is still
an interrupt. It implements `Cmd+C`, `Cmd+V`, `Cmd+T` and `Cmd+W` itself.
